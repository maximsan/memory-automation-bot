Project Memory Bot Reference Map

A compact map for setup, debugging, and choosing the right file to edit.

Runtime Flow

1. TelegramWebhook update reaches POST /api/telegram/webhook.
2. Parsesrc/core/telegram.ts turns raw payload into a small union.
3. Persistsrc/routes/webhook.ts creates a Notion note before queueing.
4. Processsrc/core/processCapture.ts calls OpenAI and edits the review message.
5. Approvesrc/core/reviewActions.ts updates project state and optional task.

Where To Change Things

Change Start here
Telegram command text or behavior src/routes/commands.ts, then tests in __tests__/telegram.test.ts.
Review card text or buttons src/core/format.ts and src/core/reviewActions.ts.
Capture parsing for text/photo/voice src/core/telegram.ts. Keep it pure and testable.
AI extraction shape src/core/schema.ts, prompts/extract-note.md, and src/integrations/openaiClient.ts.
Project state update prompt prompts/update-project-state.md and src/core/reviewActions.ts.
Notion schema docs/product-spec.md, then src/setup/notionSetup.ts and src/integrations/notionStore.ts.
Queue behavior src/integrations/queueClient.ts and app/api/queues/process-capture/route.ts.
Weekly reminder app/api/cron/weekly/route.ts, vercel.json, and config/bot.json.
Checks or deployment flow .github/workflows/checks.yml for GitHub checks, and docs/ci-cd.md for the Vercel Git deployment model.

Local Checks

  1. npm run lint for enforced style and Next/React rules.
  2. npm run typecheck for TypeScript contracts.
  3. npm test for fixture-level behavior.
  4. npm run build before deployment-facing changes.

Deployment Boundary

GitHub Actions repeats the local checks. Vercel Git integration owns deployments. Use the onboarding lesson for the concrete setup and verification path.

External State

  • .env.local holds local secrets.
  • Vercel environment variables hold deployed secrets.
  • Notion parent page owns durable databases.
  • Telegram webhook points to /api/telegram/webhook.

Development Rule

Before adding a feature, ask: does it help answer "what did I do last and what next?" If not, defer it.
